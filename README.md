# File-explorer-in-Java-
This project is a Java Swing-based file explorer application that allows users to navigate the file system and perform basic file management operations through a graphical interface.
In addition to directory navigation, the application includes built-in support for file compression and extraction using ZIP format, making it a lightweight alternative to basic file managers.
The interface is intuitive and responsive, providing users with the ability to browse drives, open folders, and manage files efficiently.

# Features
- *Browse system drives and directories*
- *Navigate forward and backward through folders*
- *Display files and directories in a structured list*
- *Open directories via selection or manual path input*
- *Compress files into .zip archives*
- *Extract files from .zip archives*
- *Simple and user-friendly graphical interface*

# Technologies Used
- *Java*
- *Swing (GUI framework)*
- *AWT*
- *Java I/O (File handling)*
- *java.util.zip (ZIP compression & decompression)*
